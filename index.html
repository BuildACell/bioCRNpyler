Installation¶

GitHub: https://github.com/BuildACell/BioCRNPyler

git clone https://github.com/BuildACell/BioCRNPyler

PyPi:

pip install biocrnpyler

What is BioCRNpyler?¶

BioCRNpyler is a Python Library and Framework. It takes high level specifications and converts them into Chemical Reaction Networks¶

BioCRNpyler is NOT a CRN Simulator¶

BioCRNpyler is an Objected Oriented Framework which Compiles CRNs¶

BioCRNpyler Specifications include Context (the Mixture), Parts (Components), and Biochemical Processes (Mechanisms)¶

Getting Started With BioCRNpyler¶

The best way to get started using BioCRNpyler is to install the software and go through the example ipython jupyter notebooks in this folder. After downloading the software, open a terminal and change directory to the examples folder. Then type:

jupyter notebook

which will open a jupyter notebook in your browser. Select the tutorial you wish to try (we recommend starting at 1. Building CRNs direction.ipynb) and learn a new, faster way of producing biochemical models!
